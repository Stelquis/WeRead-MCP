# 贡献者公约行为准则

## 我们的承诺

作为社区成员、贡献者和领导者，我们承诺让所有参与者都能在不受骚扰的环境中参与我们的社区，无论其年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

## 我们的标准

有助于营造积极环境的行为包括：

- 对他人表现出同理心和善意
- 尊重不同的意见、观点和经历
- 给予并优雅地接受建设性反馈
- 承担责任，向受我们错误影响的人道歉
- 关注对社区整体最有利的事情

不可接受的行为包括：

- 使用性化的语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱或贬低性评论，以及人身攻击或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息
- 其他在合理情况下被认为不适当的行为

## 执行

项目维护者负责澄清和执行我们的行为标准。
如果遇到滥用、骚扰或其他不可接受的行为，请向项目团队报告。
所有投诉都将被审查和调查。
